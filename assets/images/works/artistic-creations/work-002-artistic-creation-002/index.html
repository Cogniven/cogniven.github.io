Overview

This is one of my earliest works. It records an initial attempt made under a personal composition rule. At the current stage, the work has only produced a lyric text with numbered notation and has not yet been developed into a complete song.

Statement

About this work

This draft belongs to an early layer of my personal creative archive. It is not presented as a finished song, but as a record of a compositional attempt: preserving the numbered notation beside the lyric text so that the melodic direction formed at that time can be reviewed.

The current version only contains the lyric text and numbered notation. It does not yet include rhythm, note length, rests, breathing marks, dynamics, or a complete performance structure. The composition rule behind this attempt and its later development will be discussed separately on this site.
