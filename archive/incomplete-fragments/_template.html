Taxonomy Notes (Unfinished)

Mar 05, 2026

A partial note set for cross-linking category systems across works, writing, and projects.

This fragment started as an attempt to normalize category language across three major sections. It proposed shared tags, but the hierarchy definition was left incomplete.

The draft is still useful because it identified overlapping labels that caused ambiguity in navigation and filtering. It also exposed where naming conventions diverged between content and route paths.

Work stopped before enforcement rules were written. The fragment is retained as a reference for future taxonomy consolidation.
