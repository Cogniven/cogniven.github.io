QR Code Artistry Research

An experiment on turning QR codes into visual material while keeping them functional, moving from manual drawing to AI-assisted image generation.

Date 2022–2024
Status Paused
Field Visual Experiment
Mode Research / Production

Experiment Note

This project explored whether a QR code could remain scannable while also becoming a visual object with its own aesthetic language.

Intent

Starting Point

The project began with a basic question: after understanding how QR codes work, could their structure be visually transformed without losing their original function?

From there, the work moved in two directions: manual drawing and later AI-assisted generation, both aimed at testing how far a QR code could be pushed toward image-making.

Research

Understanding the Code

Before making any images, the first stage was to understand QR codes themselves: what they are, how they are structured, and which parts can tolerate modification.

This stage made the later visual experiments possible, because the project depended on knowing which transformations still allowed the code to function.

Iterations

Development Stages

Stage 01

Manual Drawing Period

Around June 2022, the first direction was manual production. Using Procreate, Photoshop, and related tools, the work focused on directly drawing and reshaping QR codes into more image-like forms.

Stage 02

AI-Assisted Attempts

From 2023 onward, the project shifted toward AI tools. Stable Diffusion was tested first, but local hardware limitations made it difficult to continue. Later, Midjourney became the main tool because cloud-based generation made the process more feasible.

Observations

What Changed

The project gradually shifted from “making an artistic QR code by hand” to “testing how image systems interpret QR structures.” The focus moved away from a single object and toward a broader visual method.

At the same time, AI development accelerated quickly. What had once required individual experimentation later became much easier, and even batch-generation scripts began to appear.

Status

Where It Stopped

Because the field moved so quickly, the project was not followed further after its earlier stages. The main value now lies in the fact that it records an earlier moment of experimentation.

What remains here is not an ongoing system, but a trace of how the question was approached across different tools and technical conditions.
